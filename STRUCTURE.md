# 📋 プロジェクト構造

このリポジトリは以下の構成で自動成長します：

## ディレクトリ構成

```
zumedf-lang/-/
├── PHILOSOPHY.md           # コア哲学（不変）
├── README.md              # 今週の刺激スポット（毎週更新）
├── PROFILE.md             # あなたの背景・目標
├── projects/              # スケール化可能なプロジェクト群
│   ├── biotope/          # ビオトープシステム
│   ├── minecraft/        # マイクラ自動化
│   ├── circular-economy/ # 循環経済
│   └── ai-agent/         # このAIエージェント自体
├── discoveries/           # 毎週の自動発見
│   ├── tokyo-events/     # 東京イベント
│   ├── research/         # 論文・研究
│   └── ideas/            # 新しいアイデア
├── scripts/              # 自動実行スクリプト
├── .github/workflows/    # GitHub Actions
└── data/                 # 収集データ・分析結果
```

## 自動更新サイクル

- ⏰ **毎日**：データ収集、自動分析
- 📅 **毎週**：レポート生成、アイデア提案、プロジェクト進化
- 🌙 **毎月**：戦略レビュー、スケール化計画更新

## アウトプット形式

1. 📄 **ドキュメント**：分析・提案・戦略
2. 💻 **コード**：実装可能なツール・スクリプト
3. 🔗 **リンク集**：パートナーシップ候補、参考資料
4. 📊 **データ**：統計・分析・メトリクス
5. 💡 **アイデア帳**：スケール化案、実験提案
